Overview

GlobalPulse Terminal is a Bloomberg-style market research dashboard built for internal investment research workflows. The platform aggregates macroeconomic indicators, market news, and real-time equity quotes into a single unified interface, enabling researchers to monitor cross-asset signals and identify trade setups efficiently.

The dashboard is currently in production use for daily market analysis, with ongoing development to expand technical analysis capabilities, sector coverage, and risk modeling features.

Core Modules

FRED Economic Decoder

Multi-component macro risk scoring engine. Decodes yield curve, Sahm Rule, real rates, and term premium into a tactical risk score (0-100).

Market Flash Stream

Aggregated news feed from Finnhub with automated risk classification, geographic tagging, and Chinese translation pipeline.

Economic Calendar

Weekly auto-rolling calendar of high-impact US economic releases with estimate/actual tracking.

Live Equity Quotes

WebSocket-driven tape for 37 curated US equities across 9 sector categories, with real-time price flash animations.

Multi-Timeframe K-Line

1m / 5m / 15m / 1h / 1d candle charts with technical indicators (MA, BB, MACD, RSI) and drawing tools.

AIS Maritime Stream

Real-time vessel tracking via AIS WebSocket for shipping disruption signals integrated into macro risk scoring.

Technical Stack

  • Single-page web application — responsive frontend with bilingual UI (Traditional Chinese / English)
  • Cloudflare Workers backend — API proxy with edge caching, WebSocket bridge, and translation pipeline
  • Data sources — FRED, Finnhub, Twelve Data, AISStream, MyMemory translation API
  • Charting — Currently lightweight-charts; evaluating professional charting library upgrade for advanced technical analysis tools

Use Cases

  • Daily morning briefing — quick scan of overnight news, economic releases, and overnight equity movements
  • Macro risk monitoring — track FRED decoder score for regime changes
  • Technical analysis — multi-timeframe K-line charting with indicators and drawing tools for entry/exit timing
  • Event-driven research — economic calendar correlated with news flow and market reaction
